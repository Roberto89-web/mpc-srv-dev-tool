# Indice documentazione ERP

Benvenuto nella documentazione del framework di personalizzazione ERP.

## Moduli disponibili

- **vendite** - Gestione ordini, clienti, offerte
- **magazzino** - Movimenti, giacenze, inventario
- **contabilita** - Prima nota, partite, bilancio

## Come usare questa documentazione

Ogni modulo ha una propria cartella con i seguenti file:
- `overview.md` - Panoramica del modulo
- `personalizzazioni.md` - Guida alle personalizzazioni disponibili

## Convenzioni generali

- I nomi delle classi custom seguono il pattern `{Modulo}Custom.java`
- Ogni personalizzazione deve essere documentata con un commento Javadoc
- Le tabelle custom sul DB vanno sempre prefissati con `Y`
